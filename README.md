# Vault - Collaborative Study Platform

Vault is a production-level MERN stack application designed for students to collaborate, share resources, and solve doubts in real-time.

## 🌟 Project Highlights

- **Real-time Collaboration**: Shared whiteboards, notes, and chat powered by Socket.io.
- **Resource Management**: Structured, community-validated academic storage.
- **Gamification**: XP, Levels, and Tiers to reward community contribution.
- **Responsive Design**: Fully optimized for Desktop and Mobile devices.

## 📂 Repository Structure

- **[Backend](./Backend)**: Node.js/Express server with MongoDB and Socket.io.
- **[Frontend](./Frontend)**: React (Vite) application with Tailwind CSS.

## 🛠 Tech Stack

| Layer | Technologies |
| --- | --- |
| **Frontend** | React, Vite, Tailwind CSS, Socket.io-client |
| **Backend** | Node.js, Express, Socket.io, Multer, Nodemailer |
| **Database** | MongoDB, Mongoose |
| **Auth** | JWT, Bcrypt.js |

## 🚀 Getting Started

1. **Backend**:
   - `cd Backend`
   - `npm install`
   - Set up `.env` (see Backend README)
   - `npm start`

2. **Frontend**:
   - `cd Frontend`
   - `npm install`
   - `npm run dev`

## 🛤 Evolution of the Project

1. **Phase 1: Core Foundation**: Set up basic authentication and room management.
2. **Phase 2: Collaboration**: Implemented the real-time whiteboard and chat.
3. **Phase 3: Community Features**: Added topic-wise file uploads and the Doubt Forum.
4. **Phase 4: Gamification**: Integrated XP, Levels, and the Global Leaderboard.
5. **Phase 5: Refinement**: Mobile optimization and dedicated user dashboards.
