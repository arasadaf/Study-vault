# Vault - Collaborative Study Frontend

The frontend for **Vault** is a modern, responsive React application designed for high-performance real-time collaboration.

## 🎨 Design & Aesthetics

Vault features a **Premium Dark Aesthetic** using:
- **Glassmorphism**: Subtle translucent panels for a modern look.
- **Vibrant Gradients**: Indigo-to-Purple gradients for a tech-focused feel.
- **Inter Typography**: Clean, modern font for maximum readability.
- **Micro-animations**: Smooth transitions and hover effects for an interactive experience.

## 🛠 Tools & Technologies

- **React (Vite)**: Lightning-fast build tool and frontend framework.
- **Tailwind CSS**: Utility-first CSS for custom, responsive, and maintainable styling.
- **Socket.io-client**: Real-time communication layer for the whiteboard and chat.
- **React Router Dom**: Client-side routing for seamless navigation (Home, My Rooms, Study Rooms).
- **Lucide/Heroicons**: Premium iconography for a professional UI.

## 🚀 Key Pages & Features

- **Home Page**: Hero section with quick join, user stats, and access to the Global Leaderboard.
- **My Rooms**: A dedicated dashboard for tracking your recent study sessions.
- **Study Room**: A comprehensive collaborative space featuring:
    - **Shared Whiteboard**: Real-time drawing and brainstorming.
    - **Shared Notes**: Collaborative markdown-style note-taking.
    - **Resource Vault**: Topic-wise materials with upvote/downvote capabilities.
    - **Doubt Forum**: Community-driven Q&A with bounty rewards.
- **Gamification Dashboard**: Visual tracking of XP, Levels, and Tier ranks (Bronze → Diamond).

## 📱 Mobile Optimization

Vault is **fully mobile-responsive**, featuring:
- **Hamburger Navigation**: Clean mobile menu for small screens.
- **Toggleable Sidebars**: Study room tools collapse on mobile to maximize workspace.
- **Touch-friendly UI**: Optimized buttons and inputs for a great mobile experience.

## 🛠 Setup & Run

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Access at: `http://localhost:5173`
